# Changelog

## 1.0.4

### Added

- Gerber 导出前新增 DRC 检测。
- 检测到 DRC 问题时弹出确认框，用户可选择继续导出或取消。

## 1.0.3

### Added

- 导出页面新增“客户端设置 -> 保存路径”输入项。
- 支持两种导出模式：
  - 填写保存路径：批量导出到目标目录。
  - 路径留空：系统“另存为”逐个保存。
- 导出结果区新增更细致日志（环境状态、重试次数、耗时、错误分类）。

### Changed

- 导出流程调整为“先 BOM，再其他文件”。
- 文件命名统一为固定模板：
  - `BOM_Board1_{板名}_{YYYY-MM-DD}`
  - `Gerber_{板名}_{YYYY-MM-DD}`
  - `DXF_{板名}_{YYYY-MM-DD}_AutoCAD2007`
  - `3D_{板名}_{YYYY-MM-DD}`
  - `PickAndPlace_{板名}_{YYYY_MM_DD}`
- 批量保存链路增强，增加多种路径写入尝试（完整路径与目录 URI）。

### Fixed

- 修复导出页面在部分异常状态下无法打开的问题。
- 修复部分场景下目录/保存失败提示不清晰的问题。
- 优化 `TypeError: Failed to fetch` 场景下的重试与错误归类。
- 当批量写入不可用时，自动降级到逐个另存为，避免整次导出直接终止。

## 1.0.2

- 初始可用版本。
